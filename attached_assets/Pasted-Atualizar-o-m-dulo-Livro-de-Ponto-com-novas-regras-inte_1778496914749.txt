Atualizar o módulo “Livro de Ponto” com novas regras inteligentes de preenchimento automático e edição dinâmica.

1. Regras para Feriados Nacionais:
   Quando uma data for identificada como:

* feriado nacional;
* feriado estadual;
* feriado municipal;

e o usuário tiver ativado a opção de preenchimento automático, o sistema deve:

Nos campos das aulas/turmas:
Exemplo:

* 1ª aula;
* 2ª aula;
* turma 101;
* turma 102;

preencher automaticamente com:

```text id="4ybyy7"
*
```

No campo:

* Assinatura

preencher automaticamente com:

```text id="j8p5cg"
FERIADO NACIONAL
```

Centralizar automaticamente o texto dentro da célula.

2. Regras para Sábado e Domingo:
   Quando o dia for:

* sábado;
* domingo;

e NÃO houver evento cadastrado, o sistema deve preencher automaticamente o campo assinatura com:

```text id="4h9k1q"
********
```

O conteúdo deve ficar:

* centralizado;
* alinhado verticalmente;
* visualmente organizado.

3. Exceção para Eventos/Reuniões:
   Se o usuário marcar no preenchimento automático opções como:

* reunião;
* evento;
* planejamento;
* conselho de classe;
* formação;
* atividade escolar;

mesmo sendo sábado ou domingo:

NÃO preencher:

```text id="yb1tfm"
********
```

Deixar o campo assinatura:

* totalmente em branco;
* pronto para assinatura manual do professor.

4. Campo Dinâmico de Presença/Falta:
   Atualmente existe preenchimento automático com:

* V = presente;
* F = falta.

Remover essa lógica fixa.

Transformar os campos em:

* totalmente editáveis;
* dinâmicos;
* personalizados.

O usuário deve conseguir:

* digitar livremente;
* apagar;
* substituir;
* inserir símbolos;
* usar abreviações;
* adicionar caracteres especiais.

Exemplos permitidos:

* P;
* F;
* ✔;
* ✖;
* A;
* Licença;
* Planejamento;
* Reunião;
* textos personalizados.

5. Melhorias de Usabilidade:
   Adicionar:

* clique rápido para edição;
* edição inline;
* preenchimento em massa;
* copiar/colar entre campos;
* seleção múltipla;
* atalhos rápidos.

6. Objetivo:
   O livro de ponto deve funcionar de forma flexível e semelhante aos livros físicos reais utilizados em escolas, permitindo personalização manual e preenchimento automático inteligente.
