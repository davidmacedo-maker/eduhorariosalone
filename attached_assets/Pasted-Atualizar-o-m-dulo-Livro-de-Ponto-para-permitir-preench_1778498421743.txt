Atualizar o módulo “Livro de Ponto” para permitir preenchimento automático inteligente com símbolos personalizados, cores e edição dinâmica.

1. Regras de Preenchimento Automático:
   Nos campos das aulas/turmas:
   Exemplo:

* 1ª aula;
* 2ª aula;
* 3ª aula;
* turmas 101, 102, 201;

quando o usuário utilizar o preenchimento automático para:

* reunião;
* planejamento;
* evento pedagógico;

o sistema deve substituir o símbolo:

```text id="yn77ta"
*
```

por:

```text id="ytr4ci"
@
```

Aplicar:

* cor verde;
* destaque visual;
* centralização automática.

2. Campo Observação:
   Ao usar preenchimento automático de reunião pedagógica, preencher automaticamente o campo:

* Observações

com:

```text id="9qlj7q"
@ Reunião Pedagógica Coletiva
```

Aplicar:

* símbolo @ em verde;
* texto organizado;
* alinhamento automático.

3. Campos Dinâmicos:
   Mesmo com preenchimento automático, deixar:

* assinatura;
* frequência;
* observações;
* aulas;
* turmas;

como campos totalmente editáveis.

O usuário deve conseguir:

* apagar;
* substituir;
* editar;
* adicionar novos símbolos;
* escrever textos livres.

4. Símbolos Personalizados:
   Permitir que o usuário digite manualmente símbolos personalizados nos campos.

Exemplos:

Cumprimento de carga horária extraclasse:

```text id="bkl02n"
&
```

Aplicar:

* cor roxa;
* destaque automático.

Descrição:

```text id="2h4vfq"
Cumprimento de Carga Horária Extraclasse
```

Conselho de classe extra-turno:

```text id="2x0e6g"
CC
```

Aplicar:

* cor roxa;
* destaque visual.

Descrição:

```text id="0u6f8v"
Conselho de Classe Extra-turno
```

5. Sistema Inteligente de Legendas:
   Criar uma área automática de legendas contendo:

* símbolo;
* significado;
* cor.

Exemplo:

* @ = Reunião Pedagógica Coletiva
* & = Cumprimento de Carga Horária Extraclasse
* CC = Conselho de Classe Extra-turno

6. Regras Visuais:
   Aplicar:

* cores automáticas;
* destaque leve;
* responsividade;
* boa legibilidade na impressão;
* compatibilidade com PDF e A4.

7. Objetivo:
   Transformar o livro de ponto em um sistema flexível, moderno e semelhante aos registros escolares reais, permitindo preenchimento automático inteligente sem perder a liberdade de edição manual do usuário.
