Corrigir erro de importação de arquivos Excel no módulo de Horário Escolar.

Problema identificado:

Atualmente o sistema apresenta falha ao importar múltiplos horários de turnos diferentes.

Exemplo:

* o arquivo do turno matutino importa corretamente;
* porém ao importar o arquivo do turno vespertino:

  * ele não aparece no sistema;
  * ou substitui o horário matutino já carregado;
  * ou faz o turno da manhã desaparecer.

Isso está causando perda de dados e necessidade de reconfiguração manual.

Correções necessárias:

1. Importação Independente por Turno:
   O sistema deve permitir importar:

* Matutino;
* Vespertino;
* Noturno;

sem sobrescrever os dados já existentes.

2. Estrutura Separada:
   Cada turno deve possuir:

* armazenamento independente;
* configuração própria;
* horários separados;
* persistência individual.

3. Mesclagem Inteligente:
   Ao importar um novo arquivo:

* manter os horários já existentes;
* adicionar apenas o novo turno;
* evitar substituição automática.

4. Identificação Automática:
   O sistema deve identificar automaticamente:

* nome do turno;
* horário inicial;
* horário final;
* quantidade de aulas;
* intervalos.

5. Correção de Estado:
   Verificar possíveis problemas relacionados a:

* state management;
* sobrescrita de arrays;
* limpeza indevida de memória;
* atualização incorreta do LocalStorage/IndexedDB;
* renderização dos turnos.

6. Persistência:
   Após importar:

* todos os turnos devem continuar visíveis;
* mesmo após atualizar/recarregar a página.

7. Interface:
   Mostrar simultaneamente:

* Matutino;
* Vespertino;
* Noturno.

Sem necessidade de:

* ativar manualmente;
* reconfigurar horários;
* reiniciar o sistema.

8. Compatibilidade:
   Garantir funcionamento correto para:

* Excel (.xlsx);
* XLS;
* CSV.

9. Objetivo Final:
   O usuário deve conseguir importar vários arquivos de horários escolares sem perder dados já cadastrados, mantendo todos os turnos carregados simultaneamente no sistema.
