Passo 1: Instalar o SDK do PocketBase no Replit
Abra o Shell do seu Replit (lado direito da tela) e execute este comando para instalar a biblioteca de conexão oficial:

Bash
npm install pocketbase
Passo 2: Criar o Arquivo de Conexão (src/lib/pocketbase.ts)
Crie um novo arquivo chamado pocketbase.ts dentro da pasta src/lib/. Se a pasta lib não existir, crie-a. Este arquivo inicializa o banco.

TypeScript
import PocketBase from 'pocketbase';

// Substitua pela URL onde o seu PocketBase estará rodando.
// Enquanto estiver testando localmente, pode ser 'http://127.0.0.1:8090'
const POCKETBASE_URL = 'https://seu-pocketbase-servidor.pockethost.io';

export const pb = new PocketBase(POCKETBASE_URL);
Passo 3: Configurar as Coleções no Painel do PocketBase
Quando você abrir o painel de administração do seu PocketBase (geralmente em /sm/), vá na seção de coleções e crie a seguinte estrutura:

Coleção Nativa users:

Clique em users > engrenagem (definições) > adicione estes campos:

nome_completo (Tipo: Text)

escola_nome (Tipo: Text — Aqui você colocará "Escola Estadual Sul América" ou "Escola Estadual de Lambari")

Sua Coleção de Dados (Exemplo: horarios):

Crie uma coleção chamada horarios.

Adicione os seguintes campos:

criador (Tipo: Relation, apontando para a coleção users, não obrigatório/opcional dependendo do caso, mas ajuda).

escola_nome (Tipo: Text — Crucial para o isolamento).

dados_horario (Tipo: JSON ou os campos de texto que seu sistema usa).

Regra de Segurança (API Rules): Na aba "API Rules" da coleção horarios, mude a regra de List/Search e View para:

Plaintext
@request.auth.id != "" && escola_nome = @request.auth.escola_nome
> Isso garante nativamente no banco que o usuário só consiga listar horários se o nome da escola gravado no horário for exatamente igual ao nome da escola gravado no perfil dele.

Passo 4: Criar a Tela de Login e Cadastro (src/pages/AuthPage.tsx)
Crie este arquivo completo. Ele gerencia o cadastro (armazenando o nome completo, e-mail para recuperação, nome de usuário, senha e o nome da escola) e o redirecionamento automático pós-login:

TypeScript
import React, { useState } from 'react';
import { useLocation } from 'wouter';
import { pb } from '../lib/pocketbase';

export default function AuthPage() {
  const [, setLocation] = useLocation();
  const [isRegistering, setIsRegistering] = useState(false);
  const [loading, setLoading] = useState(false);
  const [errorMsg, setErrorMsg] = useState('');

  // Estados dos formulários
  const [email, setEmail] = useState('');
  const [password, setPassword] = useState('');
  const [fullName, setFullName] = useState('');
  const [username, setUsername] = useState('');
  const [schoolName, setSchoolName] = useState(''); // Estado para capturar a escola

  const handleLogin = async (e: React.FormEvent) => {
    e.preventDefault();
    setLoading(true);
    setErrorMsg('');

    try {
      // Autentica usando o e-mail (ou username) e senha
      await pb.collection('users').authWithPassword(email, password);
      
      // Se der certo, envia para o dashboard
      setLocation('/dashboard');
    } catch (err: any) {
      setErrorMsg(err.message || 'Erro ao fazer login. Verifique as credenciais.');
    } finally {
      setLoading(false);
    }
  };

  const handleRegister = async (e: React.FormEvent) => {
    e.preventDefault();
    setLoading(true);
    setErrorMsg('');

    try {
      const data = {
        username: username,
        email: email,
        emailVisibility: true,
        password: password,
        passwordConfirm: password, // O PocketBase pede confirmação
        nome_completo: fullName,
        escola_nome: schoolName // Atribui a escola ao perfil dele
      };

      // Cria o registro na coleção users
      await pb.collection('users').create(data);

      // Opcional: Solicitar verificação de e-mail se configurado
      // await pb.collection('users').requestVerification(email);

      alert('Cadastro realizado com sucesso! Faça login agora.');
      setIsRegistering(false); // Retorna à tela de login
      setPassword('');
    } catch (err: any) {
      setErrorMsg(err.message || 'Erro ao realizar o cadastro.');
    } finally {
      setLoading(false);
    }
  };

  return (
    <div className="min-h-screen flex items-center justify-center bg-slate-50 px-4">
      <div className="max-w-md w-full space-y-8 bg-white p-8 rounded-xl shadow-md border border-slate-100">
        <div className="text-center">
          <h2 className="text-3xl font-extrabold text-slate-900">EduHorários</h2>
          <p className="mt-2 text-sm text-slate-600">
            {isRegistering ? 'Crie sua conta administrativa' : 'Faça login para acessar os horários'}
          </p>
        </div>

        {errorMsg && (
          <div className="bg-red-50 text-red-600 p-3 rounded-md text-sm text-center font-medium">
            {errorMsg}
          </div>
        )}

        <form className="mt-6 space-y-4" onSubmit={isRegistering ? handleRegister : handleLogin}>
          {isRegistering && (
            <>
              <div>
                <label className="block text-sm font-medium text-slate-700">Nome Completo</label>
                <input
                  type="text"
                  required
                  value={fullName}
                  onChange={(e) => setFullName(e.target.value)}
                  className="mt-1 block w-full rounded-md border border-slate-300 px-3 py-2 text-slate-900 bg-white"
                  placeholder="Ex: Prof. Carlos Silva"
                />
              </div>

              <div>
                <label className="block text-sm font-medium text-slate-700">Nome de Usuário (Username)</label>
                <input
                  type="text"
                  required
                  value={username}
                  onChange={(e) => setUsername(e.target.value)}
                  className="mt-1 block w-full rounded-md border border-slate-300 px-3 py-2 text-slate-900 bg-white"
                  placeholder="Ex: carlossilva"
                />
              </div>

              <div>
                <label className="block text-sm font-medium text-slate-700">Sua Escola / Unidade</label>
                <input
                  type="text"
                  required
                  value={schoolName}
                  onChange={(e) => setSchoolName(e.target.value)}
                  className="mt-1 block w-full rounded-md border border-slate-300 px-3 py-2 text-slate-900 bg-white"
                  placeholder="Ex: Escola Estadual de Lambari"
                />
              </div>
            </>
          )}

          <div>
            <label className="block text-sm font-medium text-slate-700">E-mail para Recuperação</label>
            <input
              type="email"
              required
              value={email}
              onChange={(e) => setEmail(e.target.value)}
              className="mt-1 block w-full rounded-md border border-slate-300 px-3 py-2 text-slate-900 bg-white"
              placeholder="seuemail@provedor.com"
            />
          </div>

          <div>
            <label className="block text-sm font-medium text-slate-700">Senha</label>
            <input
              type="password"
              required
              value={password}
              onChange={(e) => setPassword(e.target.value)}
              className="mt-1 block w-full rounded-md border border-slate-300 px-3 py-2 text-slate-900 bg-white"
              placeholder="Minimo 8 caracteres"
            />
          </div>

          <button
            type="submit"
            disabled={loading}
            className="w-full flex justify-center py-2 px-4 border border-transparent rounded-md shadow-sm text-sm font-medium text-white bg-indigo-600 hover:bg-indigo-700 disabled:opacity-50"
          >
            {loading ? 'Carregando...' : isRegistering ? 'Cadastrar Perfil' : 'Entrar'}
          </button>
        </form>

        <div className="text-center mt-4">
          <button
            type="button"
            onClick={() => setIsRegistering(!isRegistering)}
            className="text-sm text-indigo-600 hover:text-indigo-500 font-medium"
          >
            {isRegistering ? 'Voltar para o Login' : 'Não tem conta? Cadastre-se agora'}
          </button>
        </div>
      </div>
    </div>
  );
}
Passo 5: Atualizar as Rotas no src/App.tsx
Garanta que seu arquivo principal use o roteamento de hash e gerencie o estado de quem está ou não logado:

TypeScript
import { Router, Route, Switch, Redirect } from "wouter";
import { useHashLocation } from "wouter/use-hash-location";
import AuthPage from "./pages/AuthPage";
import { pb } from "./lib/pocketbase";

// Componente simples para proteger rotas e carregar o sistema apenas logado
const ProtectedRoute = ({ component: Component, ...rest }: any) => {
  // O PocketBase armazena o token localmente de forma nativa e síncrona
  const isAuth = pb.authStore.isValid;

  if (!isAuth) {
    return <Redirect to="/" />;
  }

  return <Component {...rest} />;
};

export default function App() {
  return (
    <Router hook={useHashLocation}>
      <Switch>
        {/* Tela Inicial: Login e Cadastro */}
        <Route path="/" component={AuthPage} />

        {/* Sua rota atual do Dashboard envelopada na segurança do PocketBase */}
        {/* Substitua 'SeuDashboardPrincipal' pelo nome real do componente do seu sistema */}
        <Route path="/dashboard">
          <ProtectedRoute component={() => <div>O seu layout de horários vai aqui!</div>} />
        </Route>

        <Route>
          <Redirect to="/" />
        </Route>
      </Switch>
    </Router>
  );
}

Quando o gestor criar um novo horário dentro do seu dashboard, na função de salvar (enviar dados para o PocketBase), você deve ler a propriedade da escola dele assim:

TypeScript
const escolaDoUsuario = pb.authStore.model?.escola_nome;

// Enviando para a coleção 'horarios' do PocketBase
await pb.collection('horarios').create({
  escola_nome: escolaDoUsuario, // O banco amarra o dado à escola dele
