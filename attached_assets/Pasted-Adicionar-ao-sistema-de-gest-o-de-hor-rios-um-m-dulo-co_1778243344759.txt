Adicionar ao sistema de gestão de horários um módulo completo de Livro de Ponto Escolar integrado automaticamente aos horários dos professores.

Objetivo:
Criar um sistema de livro de ponto semelhante aos utilizados em escolas estaduais e municipais, integrado com:

professores;
turmas;
disciplinas;
horários;
turnos.

O módulo deve possuir:

Cadastro Integrado do Professor:
Adicionar os campos:
Nome completo;
MASP;
Data de admissão;
Tipo de vínculo:
Efetivo;
Designado;
Disciplinas;
Turmas vinculadas;
Turno;
Carga horária semanal.
Geração Automática do Livro de Ponto:
Ao selecionar:
professor;
mês;
ano;

o sistema deve gerar automaticamente:

dias do mês;
dias letivos;
horários do professor;
disciplinas;
turmas;
turnos;
quantidade de aulas.
Estrutura do Livro:
Criar tabela organizada contendo:
Data;
Dia da semana;
Horário;
Turno;
Turma;
Disciplina;
Frequência;
Presença;
Falta;
Licença;
Planejamento;
Reunião;
Observações;
Campo de assinatura.
Campos Personalizáveis:
Permitir:
adicionar/remover colunas;
alterar nomes;
definir largura;
criar linhas extras;
ativar/desativar campos.
Ocorrências:
Adicionar opções:
Presente;
Falta;
Falta Justificada;
Licença Médica;
Licença Prêmio;
Planejamento;
Reunião Pedagógica;
Conselho de Classe;
Férias;
Abono;
Outro.
Impressão Profissional:
Criar impressão:
folha A4;
aproveitamento máximo da página;
layout semelhante a documentos escolares oficiais;
cabeçalho institucional;
assinatura;
observações finais.

Utilizar:

@media print

Garantir:

sem espaços vazios excessivos;
tabelas ajustadas automaticamente;
quebra correta de páginas;
ocultar menus e botões.
Exportação:
Permitir:
exportar PDF;
exportar Excel;
backup JSON;
compartilhamento.
Relatórios:
Gerar:
livro individual;
todos os professores;
frequência mensal;
relatório de faltas;
relatório de licenças.
Layout:
Criar:
interface moderna;
dashboard administrativo;
modo claro/escuro;
responsividade completa;
suporte para celular, tablet e computador.
Integração Automática:
O sistema deve puxar automaticamente:
horários já cadastrados;
disciplinas;
turmas;
turnos;
disponibilidade do professor.
Tecnologias:
Utilizar:
HTML5;
CSS3;
JavaScript/TypeScript;
LocalStorage ou IndexedDB;
jsPDF;
SheetJS;
pdf.js.
Objetivo Final:
O módulo deve funcionar como um livro de ponto escolar real, pronto para uso administrativo e impressão profissional.
